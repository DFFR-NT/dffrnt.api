Name

Name: string
  • Defined in dffrnt.confs/index.d.ts:2126

The fullname of the Param.

Optional Aliases

Aliases: string[]
  • Defined in dffrnt.confs/index.d.ts:2158

An array of other names that will reference this Param in:

  • {@link Auth.Method.Params} / Auth.Route.Params
  • Data.Method.Params / Data.Route.Params

Example:

 
// config/endpoints.cfg.js
module.exports = () => ({
    __DEFAULTS: {
        Params: {
            MyParam: new GNParam({ Name: 'My Param', Aliases: ['YourParam'] })
        }
    },
    User: {
        Actions: {
            Friend: new RouteDB({
                Params: {
                    YourParam: true // Refers to __DEFAULTS.Params.MyParam
                }
            })
            "/": new RouteDB({
                Params: {
                    MyParam: true // Refers to __DEFAULTS.Params.MyParam
                }
            })
        }
    }
});
